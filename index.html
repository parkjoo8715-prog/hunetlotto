이벤트 개요
HU 창립 27주년 기념 사내 로또 이벤트
🎯참여 방법
본부 선택 + 이름으로 간편하게 참여합니다.
로그인이나 사번 입력 없이 누구나 쉽게 이용할 수 있어요.
🔢번호 구조
1~27번 중에서 선택하며,
5행 × 5개 = 총 25개 번호를 뽑습니다.
행 내 중복 불가, 행 간 중복은 허용됩니다.
👤참여 제한
1인 1회 참여 가능합니다.
같은 본부 + 이름으로 재접속 시 기존 번호가 조회됩니다.
📡당첨 발표
당첨번호는 사내 방송을 통해 공개됩니다.
방송 중 확인한 번호와 내 티켓을 직접 대조해 보세요!

1
STEP 1 — 본부 선택 & 이름 입력
이벤트 진입 화면 · 참여자 식별
  1. 1
    본부를 선택하세요.
    드롭다운 목록에서 내가 소속된 본부를 고릅니다.
    교육사업본부 / 교육서비스본부 / L&D테크본부 / 휴넷 L&D연구원 / 인재경영실 / 전략경영실 / 커뮤니케이션실
  2. 2
    이름을 입력하세요.
    한글만 입력 가능합니다.
  3. 3
    "참여하기" 버튼을 누르세요.
    입력한 본부+이름으로 기존 참여 여부를 확인합니다.
  4. 4
    결과에 따라 이동합니다.
    처음 참여자 → 번호 선택 화면(STEP 2)
    이미 참여한 분 → 내 티켓 화면(STEP 3)으로 바로 이동
2
STEP 2 — 번호 선택
자동 추첨 또는 수동 선택 · 5행 × 5개
💡
5행 × 5개를 모두 선택해야 참여 확정 버튼이 활성화됩니다.
A / B / C / D / E 각 행에 1~27번 중 중복 없이 5개씩 선택하세요. (행 간 중복은 허용)
  1. 1
    "자동 추첨" 탭에서 "자동으로 뽑기" 버튼을 누르세요.
  2. 2
    로또 추첨기 애니메이션이 재생되며 A~E행 5개 줄이 동시에 생성됩니다.
    드럼롤 효과와 함께 각 행이 순차적으로 공개됩니다.
  3. 3
    애니메이션이 길게 느껴지면 "Skip" 버튼으로 즉시 결과를 확인할 수 있습니다.
  4. 4
    결과가 마음에 들지 않으면 "다시 뽑기"를 눌러 새 번호를 받을 수 있습니다. 횟수 제한 없음.
  5. 5
    번호가 결정되면 "이 번호로 참여하기" 버튼을 눌러 최종 확인 후 제출합니다.
  1. 1
    "수동 선택" 탭으로 전환하면 A~E 행 목록이 표시됩니다.
    처음엔 A행이 활성화되어 있습니다.
  2. 2
    아래 번호 그리드에서 원하는 번호를 클릭하세요.
    연속으로 5개를 선택하면 A행에 순서대로 등록됩니다.
  3. 3
    A행 5개 선택 완료 시 자동으로 B행으로 이동합니다.
    이후 B~E행도 동일하게 반복합니다.
  4. 4
    행 수정이 필요한 경우: 행 목록에서 해당 행을 클릭해 전환한 뒤,
    이미 선택된 번호를 다시 클릭하면 해당 슬롯만 비워집니다.
  5. 5
    5행 모두 완료 후 "이 번호로 참여하기"를 눌러 제출합니다.
✏️
수정 예시: A행에 1, 2, 3, 4, 5가 등록된 상태에서 3을 다시 클릭하면 3이 있던 슬롯만 비워집니다. 이후 새 번호를 선택하면 빈 슬롯에 채워집니다.
🔒
제출 후 번호 변경은 불가합니다. 확인 모달에서 최종 번호를 꼭 확인한 뒤 제출하세요.
3
STEP 3 — 내 티켓 확인 & 저장
참여 완료 후 번호 확인 및 이미지 저장
🎟티켓 카드 구성
제출이 완료되면 로또 티켓 카드가 표시됩니다.
티켓에는 이벤트명 · 본부 · 이름 · 참여일자 · A~E행 번호가 포함됩니다.
  1. 1
    화면에서 내 A~E행 5줄 번호를 확인하세요.
    번호 볼은 색상으로 범위를 구분합니다. (아래 색상 기준 참고)
  2. 2
    "이미지로 저장" 버튼을 누르거나 기기의 화면 캡처 기능을 이용해 티켓을 저장하세요.
  3. 3
    방송 시청 중 공개되는 당첨번호와 내 번호를 직접 대조하세요.
    방송 링크는 화면 상단 또는 하단 버튼에서 확인할 수 있습니다.
  4. 4
    커피 혜택이 있다면 "☕ 커피 수령하기" 버튼으로 STEP 4로 이동하세요.
💾
티켓 이미지를 저장해 두면 나중에 재접속 없이 번호를 확인할 수 있습니다. 방송 전 미리 저장해 두시길 권장합니다.
4
STEP 4 — 커피 수령
이벤트 참여 커피 혜택 수령 처리
☕미수령 상태
"아직 수령하지 않았습니다"가 표시되며
"수령 완료" 버튼이 활성화됩니다.
✅수령 완료 상태
수령 처리 후에는 수령 일시가 기록되며
버튼이 비활성화됩니다.
  1. 1
    화면에서 커피 수령 상태를 확인합니다.
    화면 진입 시 DB에서 최신 수령 상태를 자동으로 불러옵니다.
  2. 2
    "수령 완료" 버튼을 누르면 확인 모달이 표시됩니다.
    "커피를 수령하셨나요? 확인 후 변경이 불가합니다."
  3. 3
    확인을 누르면 수령 처리가 완료되고 일시가 기록됩니다.
    취소를 누르면 수령 화면을 유지합니다.
  4. 4
    수령 처리 후에는 ✅ 수령 완료 상태로 전환되며 재수령이 불가합니다.
⚠️
수령 완료 후에는 취소할 수 없습니다. 실제 커피를 받으신 후에 수령 처리를 해주세요.
🔙돌아가기 버튼 안내
커피 수령 화면 하단의 버튼은 번호 확정 여부에 따라 달라집니다.

• 번호를 확정한 경우 → "🎟 내 로또 번호 확인하기" → 티켓 화면(STEP 3)으로 이동
• 번호를 아직 선택하지 않은 경우 → "🎲 번호 선택하러 가기" → 번호 선택 화면(STEP 2)으로 이동

궁금한 점을 확인하세요
불가합니다. 확인 모달에서 "확인"을 누른 순간 DB에 저장되며 변경이 불가합니다. 제출 전 최종 확인 모달에서 반드시 번호를 한 번 더 확인하세요.
탭을 닫으면 세션이 초기화되어 STEP 1에서 다시 본부와 이름을 입력해야 합니다. 하지만 DB에 저장된 번호는 그대로 유지되므로, 재입력 후 바로 내 티켓 화면(STEP 3)으로 이동됩니다.
① 이미지 저장 기능 이용
내 티켓 화면(STEP 3) 하단의 "이미지로 저장" 버튼을 누르면 티켓 카드 영역이 자동으로 캡처되어 저장됩니다. 이름·본부·A~E행 번호·날짜가 모두 포함된 이미지입니다.

② 화면 직접 캡처 후 전달
이미지 저장 버튼이 동작하지 않는 경우, 기기의 화면 캡처 기능을 직접 이용하세요.
• PC: Win + Shift + S 또는 PrtSc
• Mac: Cmd + Shift + 4로 티켓 영역 선택 캡처
• 모바일: 볼륨 다운 + 전원 버튼 동시 누름

캡처한 이미지를 담당자에게 메신저나 이메일로 전달하시면 당첨 확인이 가능합니다.
STEP 1 진입 화면 하단의 "☕ 커피만 수령하기" 버튼을 이용하세요. 본부와 이름 입력 후 해당 버튼을 누르면 번호 선택 없이 커피 수령 화면으로 바로 이동합니다.
당첨번호는 사내 방송을 통해 실시간으로 공개됩니다. 방송에서 발표된 번호와 내 티켓 번호를 직접 대조하세요. 방송 링크는 이벤트 화면 상단 또는 하단 버튼에서 확인할 수 있습니다.
